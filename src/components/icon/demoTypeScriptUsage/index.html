Icon Demo | <uif-icon>

This demo shows how to build an Angular app with TypeScript using the provided icon enum. This option makes sure you are only using Office UI Fabric supported icons.

The included icon enum is not intended to be passed directly to the directive's uif-type attribute, rather it should be used to create the string. Refer to the index.ts file in this demo to see usage.

To run this demo, you must do the following:

  • build the library in debug mode:
    $ gulp build-lib --debug
  • transpile everything (including this demo's index.ts file) to typescript:
    $ tsc -p ./
  • use webpack to create a bundle to load modules:
    webpack src/components/icon/demoTypeScriptUsage/index.js src/components/icon/demoTypeScriptUsage/bundle.js -p
