Label Demo | <uif-label>

In order for this demo to work you must first build the library in debug mode.

To render label, use the following markup: 

<uif-label>Name</uif-label>

Name

Following markup will render required label: 

<uif-label required>Name</uif-label>

Name

This markup will render disabled label: 

<uif-label disabled>Name</uif-label>

Name
