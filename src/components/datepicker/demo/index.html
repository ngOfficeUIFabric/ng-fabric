Datepicker Demo | <uif-datepicker>

In order for this demo to work you must first build the library in debug mode.
This directive depends on
  • jQuery
  • pickadate.js
Add references to jquery.js, pickadate.js, and picker.js, and picker.date.js BEFORE the ngofficeuifabric.js reference

Demos

This markup:
<uif-datepicker ng-model="value"/>
and this script: $scope.value = new Date(2016, 3, 2);

Renders this:

Value: {{value}}

This markup:
<uif-datepicker ng-model="value_2" uif-months="Jan,Feb,Maa,Apr,Mei,Jun,Jul,Aug,Sep,Okt,Nov,Dec" uif-label="Start date" placeholder="Please, find a date" />

Renders this:

Value: {{value_2}}
